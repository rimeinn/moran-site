## 使用「東風破」下載安裝

本方案兼容 [東風破](https://github.com/rime/plum/) (Plum)，使用如下命令即可直接安裝**繁體版**。

```bash
bash rime-install rimeinn/rime-moran
```

簡化字版暫不支持該方式安裝。
