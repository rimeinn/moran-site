# 衍生作品

## 呂小布：全拼九鍵輔助碼

基於魔然拆分製作的九鍵全拼輔助碼，需掛接手心輸入法使用。方法：

* 使用 `1` 隔開全拼和輔助碼
* 輔助碼是部首的全拼

例如，「輔」可以用九宮格輸入 `fu1che`。
