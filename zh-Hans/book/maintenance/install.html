安装 ​

使用「东风破」下载安装或更新 推荐 ​

本方案兼容 东风破 (Plum)，使用如下命令即可直接安装并更新：

bash
bash rime-install rimeinn/rime-moran@trad  # 传承字版
bash rime-install rimeinn/rime-moran@simp  # 简化字版

使用 Git 下载安装或更新 ​

若您主要使用魔然方案，且希望使用 Git 管理更新，可使用 rimeinn/rime-moran 仓库的 trad 和 simp 分支。master 分支仅作为方案「源代码」，不建议追踪。

分支名说明
trad传承字版完整配置
simp简化字版完整配置

稳定版 ​

稳定版经过较多测试，适合长期使用。若计划做大幅修改，或计划长期不更新，建议选用稳定版。

→ 稳定版发布页面

最新配置包直链下载 ​

TIP

该直链等价于直接从 GitHub Actions 下载最新 master 分支产物，因此总是最新的。

但该方法不等于直接下载 master 分支代码（仓库页面上点 Download）：master 分支中保存的是繁体版数据。

桌面平台 ​

macOS ​

与 鼠须管 (Squirrel) 和 fcitx5-macos 兼容。

  • 鼠须管用户目录： ~/Library/Rime
  • fcitx5-macos 用户目录： ~/.local/share/fcitx5/rime

Windows ​

与 小狼毫 (Weasel) 兼容。

Linux ​

与 ibus-rime、fcitx-rime 和 fcitx5-rime 兼容，请根据实际使用的输入法平台选择合适的 Rime 前端。

由于「魔然」和「魔然·辅筛」方案的核心功能由 Lua 实作，因此您需要安装 librime-lua 插件。如果不安装，则无法使用这两个方案，但您可以使用不依赖于 Lua 的「魔然·整句」和「魔然·字词」。

依赖的版本信息：

  • librime >= 1.8.5
  • librime-lua >= 2023 年 8 月
  • librime-octagram (可选)

部分已知可用的发行版及其插件名：

移动平台 ​

Android ​

与 fcitx5-android 和 同文输入法 兼容。

已知与「中文输入法」（同文无障碍版）不兼容。

iOS ​

与 iRime 和 仓输入法 兼容。

本方案已内置于仓输入法的方案列表，您可以在方案列表界面中点击「方案下载」，然后就可以直接安装「魔然〈繁〉」或「魔然〈简〉」。

纯血鸿蒙 ​

已知有基于 Rime 引擎的「超越输入法」，是否兼容尚未知。
